Install Chart

Deploy PostgreSQL with default production settings:

helm install my-postgres postgresql/postgresql -n database --create-namespace

Or customize with your own values:

helm install my-postgres postgresql/postgresql -n database --create-namespace \ --set persistence.enabled=true \ --set resources.requests.memory=2Gi

Usage Examples

Connect to PostgreSQL

# Get connection details export POSTGRES_PASSWORD=$(kubectl get secret my-postgres-secret -o jsonpath="{.data.POSTGRES_PASSWORD}" | base64 -d) # Port forward to access locally kubectl port-forward service/my-postgres-service 5432:5432 # Connect using psql psql -h localhost -U konga -d konga

Backup & Restore

# Create backup kubectl exec -it my-postgres-0 -- pg_dump -U konga konga > backup.sql # Restore from backup kubectl exec -i my-postgres-0 -- psql -U konga -d konga < backup.sql

Uninstall Chart

helm uninstall my-postgres -n database

Note: This will not delete persistent volumes. Delete them manually if needed.
